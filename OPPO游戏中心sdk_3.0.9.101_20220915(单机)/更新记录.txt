4、2.0.8.010版本更新注意事项：

新版本需要引入 androidx的v4包 implementation 'androidx.legacy:legacy-support-v4:1.0.0'，不方便引入的用SDK压缩包中提供的core-1.1.0代替。
编译的时候需要加一下
    compileOptions {
        sourceCompatibility JavaVersion.VERSION_1_8
        targetCompatibility JavaVersion.VERSION_1_8
    }
